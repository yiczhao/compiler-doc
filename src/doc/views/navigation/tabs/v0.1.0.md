### API

| 参数 | 说明 | 接口类型 | 类型 | 双向（twoWay） | 是否必须 | 可选值 | 默认值 |
|------|-------|----------|---------|---------|---------|-------|--------|
| type | KsTabs类型 | props | String | `false` | 否 | `overline``underline``border` | `overline` |
| v-on:change | tab改变时触发 | props | Event | `false` | 否 |  |  |
| status | 当前状态(选中) | props | String | `false` | 否 | `active` |  |
